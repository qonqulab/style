## Phase 2: Core Style Implementation
Each style includes a preview card, a detailed explanation, and its pros/cons.

- [x] **1. Glassmorphism**
    - Description: Frosted glass effect using backdrop-filter.
    - Pros: Elegant, depth, modern.
    - Cons: High GPU usage, accessibility challenges.
- [x] **2. Neumorphism**
    - Description: Soft UI using inner and outer shadows.
    - Pros: Tactile, clean, unique.
    - Cons: Hard to achieve good contrast, accessibility.
- [x] **3. Claymorphism**
    - Description: 3D-like inflated look with soft inner shadows.
    - Pros: Playful, friendly, depth.
    - Cons: Can look unprofessional if overused.
- [x] **4. Neo-Brutalism**
    - Description: Bold, raw, high-contrast borders and clashing colors.
    - Pros: High impact, memorable, honest.
    - Cons: Can be visually overwhelming.
- [x] **5. Pink Pastel / Cute**
    - Description: Soft colors, rounded corners, playful typography.
    - Pros: Friendly, calming, trendy.
    - Cons: Niche appeal.

- [x] **6. Minimalist**
    - Description: Focus on whitespace and essential typography.
    - Pros: Fast loading, high readability, timeless.
    - Cons: Risk of looking "boring" or "empty".
- [x] **7. Dark Futuristic / Cyberpunk**
    - Description: Neon glows, deep blues/purples, high tech feel.
    - Pros: Premium feel, great for tech/gaming.
    - Cons: Harder to read in high-light environments.
- [x] **8. Aurora UI / Mesh Gradients**
    - Description: Organic, colorful blurred backgrounds.
    - Pros: Dynamic, beautiful, modern.
    - Cons: Hard to implement cross-browser consistently.
- [x] **9. Retro Vaporwave**
    - Description: 80s neon, pixel art, glitches, and lo-fi vibes.
    - Pros: Nostalgic, artistic.
    - Cons: Limited professional use cases.
- [x] **10. Skeuomorphism (Modern)**
    - Description: Realistic textures and lighting.
    - Pros: Intuitive, premium.
    - Cons: Complex to design and code.
- [x] **11. Swiss Design / Editorial**
    - Description: Strict grid, asymmetric layouts, bold sans-serif.
    - Pros: Authoritative, structured, clear.
    - Cons: Requires strong content hierarchy.
- [x] **12. Organic Earth**
    - Description: Natural curves, earth tones, eco-friendly feel.
    - Pros: Grounding, sustainable vibe.
    - Cons: Hard to balance with digital constraints.
- [x] **13. Bauhaus**
    - Description: Geometric shapes, primary colors, functionalism.
    - Pros: Timeless, artistic, mathematical.
    - Cons: Can feel cold or overly rigid.
- [x] **14. Liquid / Fluid Design**
    - Description: Smooth transitions, organic moving shapes.
    - Pros: Highly engaging, "alive".
    - Cons: Performance heavy.
- [x] **15. Flat Design 2.0**
    - Description: Flat with subtle shadows for depth.
    - Pros: Practical, fast, universal.
    - Cons: Common/generic.
